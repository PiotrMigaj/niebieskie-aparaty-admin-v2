## SAM CLI + OrbStack

SAM CLI 1.144's bundled Python `docker` library negotiates API v1.35; OrbStack requires ≥1.40, causing `sam build --use-container` to fail with "Docker is unreachable". Workaround: `export DOCKER_API_VERSION=1.45`. Or skip the container build and rely on the cross-platform npm_config trick in `selection-serverless/`'s `pnpm build` script.
